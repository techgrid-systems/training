# Power Automate Training Platform

Welcome to the Power Automate Training Academy repository! This platform is dedicated to upskilling professionals in Microsoft Power Automate, ranging from foundational automation to complex enterprise lifecycle management. 
<br>

Project Overview:
This repository contains the source code for a structured, professional training portal designed to guide users through the automation lifecycle. The platform features a dual-layered navigation system, a clean user interface, and clear progression paths for learning.Certification PathThe platform is organized into three distinct levels of expertise, allowing students to progress logically:
<br>

PA 1 (Foundational Automation): Covers basic trigger configurations (e.g., Microsoft Forms integration) and an introduction to the Power Automate interface.
<br>
PA 2 (Intermediate Integration): Focuses on handling complex JSON payloads, filter components, and third-party service integration (e.g., Cloud Radial).
<br>
PA 3 (Enterprise Architecture): Advanced professional services automation, covering API-driven ticket management, Autotask integration, and automated lifecycle management.Key Features
<br>
Structured Learning: Clearly defined certification levels (PA 1, PA 2, PA 3).
<br>
Resource Hub: Downloadable source documentation and automation templates.Dual-Layered Navigation: Intuitive UI design for easy access to training modules, support, and resources.
<br>
Responsive Design: Built with Tailwind CSS for a modern, professional look across all devices.Technology StackHTML5Tailwind CSS (for styling and layout)Vanilla JavaScript (where applicable)
