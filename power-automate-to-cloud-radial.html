1. HTTP Request Trigger

Set up the When an HTTP request is received trigger using the following JSON payload schema[cite: 1, 2]:

{
  "type": "object",
  "properties": {
    "type": { "type": "string" },
    "properties": {
      "type": "object",
      "properties": {
        "Question": {
          "type": "object",
          "properties": { "type": { "type": "string" } }
        },
        " Question ": {
          "type": "object",
          "properties": { "type": { "type": "string" } }
        }
      }
    }
  }
}

2. Data Filtering

Create separate filter components for your questions[cite: 27]:

  • From: triggerBody()?['Ticket']?['Questions'] [cite: 27]
  • Filter Query: item()?['Id'] [cite: 27]
  • Is Equal To: The JSON ID (e.g., requestBy) [cite: 27]

3. Compose Functions

Create the following Compose functions to structure your data[cite: 28]:

  • Job Title: first(body('Filter_JobTitle'))?['value'] [cite: 28]
  • First Name: body('Filter_First_Name')?[0]?['Value'] [cite: 28]
  • Last Name: body('Filter_Surname')?[0]?['Value'] [cite: 28]
  • Display Name: concat(outputs('Compose_First_Name'), ' ', outputs('Compose_Last_Name')) [cite: 28]
  • UPN: concat(toLower(coalesce(outputs('Compose_First_Name'), '')), '.', toLower(coalesce(outputs('Compose_Last_Name'), '')), '@yourdomain.com') [cite: 28]
  • Temporary Password: concat(substring(base64(guid()), 0, 9), '1', '!', 'A') [cite: 28]

4. Create User (Entra/AD)

Configure the Create User action with these parameters[cite: 29, 30]:

  • Account Enabled: yes [cite: 30]
  • Display Name: outputs('Compose_Display_Name') [cite: 30]
  • Mail Nickname: first(split(outputs('Compose_-_User_Principal_Name_(UPN)'), '@')) [cite: 30]
  • Password: outputs('Compose_-_Temporary_Password') [cite: 30]
  • User Principal Name: outputs('Compose_-_User_Principal_Name_(UPN)') [cite: 30]
  • Given Name: outputs('Compose_First_Name') [cite: 30]
  • Surname: outputs('Compose_Last_Name') [cite: 30]
  • Advanced - Job Title: outputs('Compose_JobTitle') [cite: 30]
  • Advanced - Office Location: first(body('Filter_OfficeLocation'))?['Value'] [cite: 30]

5. Email & Manual Intervention

Compose an email body detailing the following[cite: 32]:

  • Name/Email/Pass: Use the outputs from the Create User and Compose steps[cite: 36, 37, 38].
  • Employee ID/Office: Use first(body('Filter_Postcode'))?['Value'] and first(body('Filter_OfficeLocation'))?['Value'][cite: 39, 41].
  • Manual Tasks: Include instructions to add the Manager (first(body('Filter_ManagerEmail'))?['Value']), assign to groups, add to shared mailboxes, and assign Microsoft Licenses[cite: 44, 45, 46, 47].
