Power Automate to Microsoft Forms Integration Guide

1. Trigger & Response Details

  • Trigger: When a new response is submitted[cite: 2].
  • Instruction: Select the form where you want the answers to come from[cite: 2].
  • Get response details: Form ID = the form name[cite: 2].
  • Response ID: triggerOutputs()?['body/resourceData/responseId'][cite: 2].

2. Variables

  • Initialize variable (First Name): Name: varFirstName, Type: String, Value: Question First Name[cite: 2].
  • Initialize variable (Last Name): Name: varLastName, Type: String, Value: Question Last Name[cite: 2].

3. Compose Functions

  • User Principal Name (UPN): concat(variables('varFirstName'), '.', variables('varLastName'), '@yourdomain.org')[cite: 2].
  • Display Name: concat(variables('varFirstName'), ' ', variables('varLastName'))[cite: 2].
  • Temporary Password: concat(substring(base64(guid()), 0, 9), '1', '!', 'A')[cite: 2].

4. User Provisioning

  • Create user: Account Enable = Yes[cite: 2].
  • Display Name: outputs('Compose_-_Display_Name')[cite: 2].
  • Mail Nickname: first(split(outputs('Compose_-_User_Principal_Name_(UPN)'), '@'))[cite: 2].
  • Password: outputs('Compose_-_Temporary_Password')[cite: 2].
  • User Principal Name: outputs('Compose_-_User_Principal_Name_(UPN)')[cite: 2].
  • Given Name: Question First Name (Dynamic Content)[cite: 2].
  • Surname: Question Last Name (Dynamic Content)[cite: 2].
  • Job Title: Question Job Title (Dynamic Content)[cite: 2].
  • Update user: User Id or Principal Name = outputs('Create_user')?['body/userPrincipalName'][cite: 2].
  • Additional Properties: employeeId = outputs('Compose_-_Postcode_Employee_ID')[cite: 2].
